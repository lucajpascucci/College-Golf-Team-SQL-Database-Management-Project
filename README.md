# College-Golf-Team-SQL-Database-Management-Project
Built a relational SQL database in Microsoft Access to manage college golf team roster, tournament, and performance data. Designed normalized tables with primary and foreign keys, wrote basic SQL queries using joins and aggregates, and documented the schema with text file exports and screenshots.

## Objectives
- Organize sports performance data using relational tables
- Practice SQL querying and aggregation
- Ensure data accuracy and integrity

## Tools
- Microsoft Access
- SQL (basic)
- CSV exports for portability

## Features
- Normalized tables with primary and foreign keys
- SQL queries using JOIN, GROUP BY, and aggregate functions
- Relationship diagram included
- Sample reports and queries

## Notes
The original .accdb file is not included. All tables, queries, and schema
are documented here for review.
